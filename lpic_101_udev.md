# Тема: работа с железом в Linux. (LPIC 101.1)

## Udev - основные команды.

1.mem - слепок оперативной памяти.
2.null - нулевое устройство (можно исп. как адресат для удаления ненужного).
3.urandom - генератор случайных чисел.
4.sdx - носители информации
5.pts - псевдо-терминал
6.tty - подсистема для общения пользователя с системой через текст (терминал)
